Life's Dashboard
A dynamic personal dashboard built as part of a supervised internship project.
What it does
The user lands on a questionnaire page and answers a series of questions one at a time. Once submitted, the answers are injected into a personal dashboard displaying their personal and work information.
How it works

Multi-step questionnaire with Next, Previous and Submit navigation
Input validation on each question
Answers are passed from the questionnaire to the dashboard using localStorage
Dashboard reads and displays the answers, then clears localStorage

Built with

HTML
CSS
JavaScript
FontAwesome for icons
